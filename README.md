# Jet Engine Simulation

## Overview
This project simulates jet engine operations including thermodynamic and aerodynamic calculations. It provides insights into jet engine performance and efficiency through a variety of computational models implemented in Julia.

## Features
- Thermodynamic cycle analysis
- Aerodynamic performance simulation
- Visualization of engine parameters
- Modular architecture for easy enhancements

## Installation
To install this project, ensure you have Julia installed on your machine, then clone the repository and navigate to the project directory:

```bash
git clone https://github.com/reansuens/JetSimulation.git
cd JetSimulation
```

## Usage
After installation, you can run the simulation using the Julia command line (enter Julia by simply typing ```julia``` in your terminal emulator:

```julia
include("main.jl)
```

## Technical Details
This simulation uses mathematical models based on the Brayton cycle and various aerodynamic principles. The computations are designed for efficiency and accuracy, leveraging Julia's capabilities for high-performance numerical methods.

## Simulation Pipeline
1. **Input Parameters:** Define engine parameters such as pressure, temperature, and fuel type.
2. **Model Initialization:** Load the thermodynamic and aerodynamic models.
3. **Solver Execution:** Execute the simulation solver to compute outputs.
4. **Results Analysis:** Gather and visualize simulation results.

## Performance Metrics
- **Thrust (N):** The amount of force generated by the engine.
- **Specific Fuel Consumption (SFC):** Measure of the fuel efficiency.
- **Thermal Efficiency (%):** Efficiency of converting fuel energy into thrust.
